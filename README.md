使用说明：
1.下载并完成解压
2.前往：https://app.extended.exchange/join/ZAIJIN 新建API
3.点击 Show API Detalls
4.保存EXTENDED_API_KEY、EXTENDED VAULT、EXTENDED STARK PRIVATE KEY、EXTENDED STARK PUBLIC KEY
5.用记事本方式打开文件中的.env文件，填入刚才保存的API信息
6.双击实盘启动开始运行
7.在网页端选择交易对、K线周期然后刷新趋势
系统会判断最新最近代币走向给出当前行情趋势
8.选择网格类型（可以参考系统分析的趋势来选择趋势类型）
9.选择稳健或者激进风格
10.点击根据当前行情智能填充参数
下方会根据你的账户余额，自动填入合适的数据，也可自动调整你的参数
注意：调整倍数后每格数量也会发生变化，启动之前需要再核对一下参数
11.点击启动机器人，程序开始运行

注意事项：
Extended API在香港环境无法运行，如果实盘启动失败可以先检查一下你的VPN是不是连接了香港
网格交易遇到单向行情还是会亏损，注意控制风险
交易程序默认：代币价格冲破区间之后会自动停止+撤单+平仓。
本程序仅供模拟使用~如用于实盘交易风险自负~
